JavaScript Guide
Introduction Variables Functions Loops Arrays
Introduction

JavaScript is a programming language used for web development.

It allows you to create interactive websites.

console.log("Hello World");
  • Easy to learn
  • Runs in browser
Variables

Variables store data values.

You can declare variables using let, const, or var.

let name = "John"; const age = 25;
  • let is block scoped
  • const cannot be reassigned
Functions

Functions are reusable blocks of code.

They help organize your program.

function greet() { return "Hi"; }
  • Reusable
  • Improve readability
Loops

Loops repeat code multiple times.

Common loops include for and while.

for(let i=0; i<5; i++) {}
  • for loop
  • while loop
Arrays

Arrays store multiple values.

They are ordered collections.

let arr = [1,2,3];
  • Indexed
  • Flexible size
